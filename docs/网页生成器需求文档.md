# 网页生成器模块 · 需求文档

> 文档版本：v1.0 · 2026-04-23

---

## 一、产品概述

**模块名称**：一键生成动画视频
**用户路径**：用户粘贴任意文章/文字 → AI 提炼精华  → 生成可分享的视频总结

**核心价值**：把长内容压缩成 16:9 横屏短视频，适合抖音/视频号/小红书传播。

---

## 二、输入输出规范

### 输入

| 字段 | 类型 | 约束 |
|------|------|------|
| text | 字符串 | 20 ~ 8000 字，文章链接或纯文字均可 |

### 输出（AI 提炼结构）

```json
{
  "title": "核心标题（≤12字）",
  "points": [
    {
      "label": "核心词（2-4字，有冲击力）",
      "short": "一句话补充（5-10字）",
      "desc": "详细解释（15-30字）",
      "formatted": "精炼格式（3-5字事件：3-5字感悟，共10-16字，供中国风样式专用）"
    }
  ]
}
```

### 字段规则

- 若原文有明确维度（如"三维度"、"三个要点"），严格对应生成，不增不减
- 若原文无明确分类，自然分 3~5 个主题点
- AI 模型：DeepSeek Chat，温度 0.3，max_tokens 1024
- API 密钥从环境变量 `DEEPSEEK_API_KEY` 或 `VITE_DEEPSEEK_API_KEY` 读取

## 四、VideoGenerator 动画引擎规范

### 画布规格

- 分辨率：1920 × 1080（16:9 竖屏）
- 预览缩放：340 × 604（CSS 缩放）

### 动画时间轴

| 阶段 | 时长 | 说明 |
|------|------|------|
| 背景就绪 | 0~600ms | 粒子 + 网格 + 光晕渐入 |
| 标题打字机 | 600ms ~ titleEnd | 逐字打出，每字 90ms，最后一字弹性动画 |
| 标题上移 | ~+600ms | 从画面中心缓动上移至顶部（100px），字号 60→40 |
| 内容卡片 | titleMoveEnd+200ms 开始 | 每张卡片入场 400ms，间隔 2400ms |
| 结尾 | contentEnd 开始 | 淡入遮罩 + 标题重现 + 品牌水印 |
| **总时长** | **动态计算** | 约 `600 + titleLen×90 + 1800 + pts.length×2400 + 2500` ms |

### 视觉规范

**背景层**：

- 线性渐变（3 色）
- 中心径向光晕（主题色，透明度渐变）
- 浮动粒子（带脉冲动画）
- 透视网格（随高度透明度递减）

**标题**：

- 打字机阶段：居中，60px，字重 900，带光标闪烁
- 上移阶段：缓动曲线 `easeOutCubic`
- 稳定阶段：顶部 100px，40px，下方装饰横线

**内容卡片**（每张）：

- 玻璃态背景（`rgba(255,255,255,0.05)` 填充 + 主题色边框）
- 左侧彩色竖线（主题色，圆角）
- 序号圆圈：居左，40px 半径，主题色填充，白色数字（36px）
- 小标题：序号右侧，38px，字重 800，主题色
- 说明文字：标题下方，26px，70% 白色，自动换行
- 装饰图形：右侧动态脉冲圆 + 旋转三角，左上角入场装饰线

**结尾页**：

- 半透明主题色遮罩渐入
- 标题居中重现（52px）
- 副标题"— 小福AI自由 —"（24px，50% 白色）

**水印**：

- 右下角，`@小福AI自由`，主题色 25% 透明度，20px

### 视频录制

- 技术：`canvas.captureStream(30)` + `MediaRecorder`
- 格式：`video/webm;codecs=vp9`（回退 `video/webm`）
- 分块：100ms 间隔
- 下载文件名：`{title前10字}.webm`

### 主题系统（VideoGenerator 专用）

| Key | 名称 | 粒子色 |
|-----|------|--------|
| `deep-space` | 深空蓝 | `#00d4ff` |
| `midnight-gold` | 暮夜金 | `#f0c040` |
| `aurora` | 极光紫 | `#a855f7` |

## 五、UI 流程

```
第一步：选择风格
  └─ 中国风 / 城市地标 / AI科技（三选一）

第二步：配置内容
  ├─ 粘贴文字（textarea，必填，20字起）
  ├─ 封面选择（根据风格显示对应选项）
  ├─ 高级选项（中国风可选展开）
  └─ 点击「生成并录制视频 →」

第三步：AI 提炼 → 自动进入 VideoGenerator 全屏录制
  ├─ 预览动画 / 录制视频（两个按钮）
  ├─ 录制完成 → 下载视频 / 重新录制
  └─ 关闭返回首页
```

## 六、错误处理

| 场景 | 错误信息 |
|------|----------|
| 内容 < 20 字 | `内容太短，请输入至少20个字符` |
| 内容 > 8000 字 | `内容过长，请控制在8000字以内` |
| API Key 未配置 | `DeepSeek API 未配置，请联系站长` |
| API 返回非 200 | `DeepSeek API 错误: {status}` |
| AI 返回空内容 | `AI返回内容为空` |
| AI 返回格式非法 | `AI返回格式错误，请重试` |
| 数据结构不符合 | `AI返回数据格式不符合要求，请重试` |
| 网络错误 | `网络错误，请重试` |

## 七、技术约束

- 框架：Vite + React + Tailwind CSS
- 部署：Vercel（API 用 Serverless Functions）
- AI：DeepSeek Chat API（`deepseek-chat` 模型）
- 视频：纯客户端录制，无需服务端转码
- 主题持久化：localStorage
