# 小福 · 人生控制台

> 少工作，多赚钱，以书为粮，以路为行。

这不是一个普通的作品集网站。这是我人生的控制台。

---

## 一、我是谁

我叫小福，甘肃深山人。

从小放牛、捡粪、烧土豆，靠读书走出大山，考上大学，拿到硕士。

母亲不识字，一天学没上过，照顾爷爷三十年。爷爷瘫痪前一直在干活，挖药种地，说"给孙子挣娶媳妇的钱"，可到走都没看到孙子娶上媳妇。

这些经历塑造了我：真实、朴素、不回避困惑和失败。

**座右铭：** 少工作，多赚钱，以书为粮，以路为行。

**理想生活：** 主业教师 + 副业AI + 自媒体创作者 + 自己的小书店 + 健身房 + 阅读跑步贯穿一生。

---

## 二、这个网站是什么

**一个中心，两条并行线。**

- **中心：** 我是谁（头像、自我介绍、座右铭）
- **线一：读书** — 阅读记录、年度进度、读书笔记
- **线二：跑步** — 运动轨迹、跑步记录、心得

中间连着作品和社交：我在做什么，去哪里找我。

这个网站不是表演，是记录。记录真实成长，不装完美。

---

## 三、功能模块

### 首页
- Hero 区：头像 + 自我介绍 + 座右铭
- 作品轮播：展示核心项目
- 社交平台数据：抖音/视频号/快手/公众号
- 读书速览：年度进度 + 当前在读
- 跑步速览：核心数据 + 最近记录
- 联系方式：二维码 + 社交链接

### 作品页
- 分类筛选 + 搜索 + 标签
- 作品详情页（/works/:id）
- 封面图展示

### 阅读页（/reading）
- 年度阅读进度（环形图）
- 月度阅读量（柱状图）
- 当前在读 + 进度条
- 阅读记录列表 + 读书笔记

### 跑步页（/exercise）
- 运动轨迹地图（中国地图 SVG）
- 核心数据（总距离/总时长/总次数）
- 跑步记录列表 + 心得

### 后台管理（/admin）
- 7个模块配置：Hero / 社交 / 作品 / 阅读 / 跑步 / 关于 / 主题
- 阅读记录 & 跑步记录：新增 / 编辑 / 删除
- AI 写作助手：每条记录可展开 AI 写作面板，辅助生成读书笔记/运动心得
- 图片上传：头像 / 封面 / 二维码（base64 存储，1MB 限制）
- 主题切换：dark / light

## 四、技术栈

| 技术 | 用途 |
|------|------|
| React 19 + Vite | 前端框架 + 构建工具 |
| React Router | 路由管理 |
| Tailwind CSS | 样式系统 |
| Lucide React | 图标库 |
| localStorage | 配置持久化（siteConfig） |
| DeepSeek API | AI 写作（后台集成，本地降级兜底） |
| Vercel | 托管部署 |

---

## 五、设计原则

1. **后台创作，前台展示** — 创作在 Admin，访客只看成品
2. **配置驱动** — 数据与展示分离，siteConfig 统一管理
3. **真实 > 完美** — 不装、不演、不编，朴素但有血有肉
4. **最小代价，最快实施** — 先跑起来，再迭代
