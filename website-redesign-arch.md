# 网站重构架构设计文档
> 版本：v1.0 | 日期：2026-05-05

---

## 一、设计理念

**核心理念**：展示型 + 管理型双模式网站
- 访客看到的是精炼的作品展示
- 管理员通过 `/admin` 进入后台管理作品内容
- 所有内容通过 Supabase 管理，数据驱动

---

## 二、页面结构总览

```
/ (首页)           → 纯个人介绍，快速传达价值
/works (作品集)     → 左侧分类筛选 + 右侧网格卡片
/journey (心路历程)  → 读书/运动/自媒体 三板块数据可视化 + 时间轴
/contact (联系)     → 联系方式 + 社交链接

/admin (管理后台)  → 作品管理 / 内容创作 SOP 入口（仅管理员）
```

## 四、功能交互规范

### 4.1 页面切换
- 路由切换时整体 fade-out → fade-in（200ms）
- 进度条：顶部细线，页面加载时填充动画

### 4.2 导航行为
- 滚动超过 80px：Nav 背景变为毛玻璃效果（backdrop-filter: blur(12px)）
- 当前页面链接高亮（底部下划线指示器）
- 移动端：Nav 折叠为汉堡菜单，全屏覆盖导航

### 4.3 卡片交互
- Hover：阴影加深 + 微上浮
- Click：轻微缩小反馈（scale(0.98)）
- 加载中：骨架屏占位

### 4.4 表单交互
- 输入框 focus：边框高亮 + 轻微放大
- 错误：红色边框 + 下方错误提示文字
- 提交：按钮 loading 状态（spinner），防重复提交

### 4.5 图片处理
- 所有图片懒加载（Intersection Observer）
- 比例容器（aspect-ratio）防止布局抖动
- 截图支持点击放大（lightbox）

## 六、技术架构

### 6.1 技术栈
- **框架**：Vite + React 18 + React Router v6
- **样式**：Tailwind CSS v3
- **数据可视化**：Recharts
- **后端**：Supabase（PostgreSQL + Storage）
- **部署**：Vercel

### 6.2 数据库设计（Supabase）

```sql
-- 作品集表
CREATE TABLE works (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title       TEXT NOT NULL,
  category    TEXT NOT NULL CHECK (category IN ('AI应用','全栈','工具','数据','自媒体')),
  status      TEXT DEFAULT '草稿' CHECK (status IN ('草稿','已发布')),
  cover_url   TEXT,
  screenshots TEXT[], -- 多张截图数组
  description TEXT,
  tech_stack  TEXT[],
  source_url  TEXT,
  live_url    TEXT,
  order_index INT DEFAULT 0,
  created_at  TIMESTAMPTZ DEFAULT NOW(),
  updated_at  TIMESTAMPTZ DEFAULT NOW()
);

-- 心路历程表
CREATE TABLE journey_entries (
  id         UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  category   TEXT NOT NULL CHECK (category IN ('读书','运动','自媒体')),
  title      TEXT NOT NULL,
  content    TEXT,
  happened_at DATE NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 统计数据表
CREATE TABLE journey_stats (
  id         UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  category   TEXT NOT NULL UNIQUE,
  total      INT DEFAULT 0,
  monthly    INT DEFAULT 0,
  streak     INT DEFAULT 0,
  last_date  DATE,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 联系表单记录
CREATE TABLE contact_messages (
  id         UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name       TEXT NOT NULL,
  email      TEXT NOT NULL,
  message    TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
