# front-end-skills

Comprehensive toolkit for AI agents to handle front-end development tasks.

## Abilities

1. **React Component Development** - Create reusable React components with TypeScript, hooks, state management
2. **Full Web Page Building** - Build complete web pages from wireframes or descriptions
3. **Node.js Backend** - Develop Express/Koa/Fastify backends, REST APIs
4. **Mini-program Development** - WeChat mini-programs, Alipay mini-programs
5. **Styling** - Tailwind CSS, Sass, CSS-in-JS, responsive design
6. **Documentation** - MDX, Storybook, API docs
7. **Browser Automation** - Playwright, Puppeteer, E2E testing
8. **SEO Optimization** - Meta tags, structured data, performance
9. **Image & Resource Management** - Optimization, sprites, icons
10. **Automated Testing** - Jest, React Testing Library, Cypress

## Workflow

When a front-end task is triggered:

1. **Analyze** - Understand requirements, check project structure
2. **Plan** - Choose tech stack, estimate complexity
3. **Implement** - Write code following best practices
4. **Test** - Verify functionality, check for errors
5. **Optimize** - Performance, accessibility, SEO

## Tech Stack Coverage

- React 19, Next.js 14, Vite
- TypeScript, JavaScript (ES6+)
- Tailwind CSS, Sass, CSS Modules
- Node.js, Express, Fastify
- Playwright, Puppeteer, Cypress
- Jest, React Testing Library
- Lighthouse, Web Vitals

## Usage

When user describes a front-end task, load this skill and apply its workflow to implement the solution.
